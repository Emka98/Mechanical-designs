# Moje Projekty CAD

To repozytorium zawiera zbiór moich hobbystycznych modeli CAD. 
Projekty tworzone są hobbystycznie w czasie wolnym.

### Co tu znajdziesz:
* Pliki źródłowe projektów.
* Modele wyeksportowane do formatów uniwersalnych DWG SLDPRT .
* Proste konstrukcje, części zamienne i prototypy.

---
*Modele udostępniam w celach poglądowych lub do własnego użytku.*
